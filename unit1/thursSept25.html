HTML Lesson 3: Text Formatting, Ordered Lists, and Unordered Lists

Basic Text Formatting

Use the strong tag or the bold tag to make something bold.

The Difference Between Strong and B

The Strong tag is an HTML Semantic tag. There is a meaning behind using the strong tag. It tells the browser there is something important about the text inside that tag.

The B tag, however, means nothing. It simply bolds text.

The em Tag and the i Tag

The em tag and the i tag make the text inside of them italic. Again, the difference is that the em tag is a semantic tag. The i tag simply turns text italic.

The u Tag

The u tag simply underlines the text inside it.

HTML Lists

There are 3 types of HTML lists: an ordered list, an unordered list, a description, or a dictionary list.

The Ordered List

  1. Item 1
  2. Item 2
  3. Item 3
  4. Item 4

The (un)Ordered List

  • Item 1
  • Item 2
  • Item 3
  • Item 4

Nested Lists

  • Item 1
    • Nested 1
    • Nested 2
    • Nested 3
  • Item 2
  • Item 3
  • Item 4
